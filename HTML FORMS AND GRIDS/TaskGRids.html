The Grid Row

 An element that allows you to 
build interesting layout grids
Grid rows are ful width
elements and you can place 
any number of rows below
      each other 
    
You can set background color and image 
    for each cell individually    
A row can hold upto 5 cells
    that are aligned 
    beside each other
    
Use your imagination when it comes to using this element and you can build really 
 interesting stuff
    
        As shown below (the 4 blue bars)you can even use the elements for pure styling
